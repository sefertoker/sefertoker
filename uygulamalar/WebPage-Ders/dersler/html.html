HTML Dersleri

Web geliştirmenin temel yapı taşı olan HTML'i öğrenmeye başlayın. Bu kapsamlı derste, modern web sayfaları oluşturmak için ihtiyacınız olan tüm HTML bilgilerini edineceksiniz.

Örnek HTML Kodu:

HTML Nedir?

HTML (Hypertext Markup Language), web sayfalarının yapısını oluşturmak için kullanılan standart işaretleme dilidir. Web tarayıcıları bu işaretlemeleri okuyarak içeriği kullanıcılara görsel olarak sunar.

Temel HTML Etiketleri

  • <html> - HTML belgesinin kök elementi
  • <head> - Belge başlığı ve meta bilgileri
  • <body> - Belgenin görünen kısmı
  • <h1> - <h6> - Başlıklar
  • <p> - Paragraf
  • <div> - Bölüm
  • <span> - Satır içi element
